FEATURES E PLANOS FUTUROS - SIMULADOR DE LEITO DE EXTRAÇÃO
=========================================================

1. FEATURES IMPLEMENTADAS
------------------------

1.1 Sistema de Câmeras
---------------------
✓ Sistema de múltiplas câmeras (Front, Free, Iso, Top)
✓ Controles independentes para cada câmera
✓ Ajuste automático de FOV baseado no tamanho do leito
✓ Movimento vertical mantido para câmeras Front e Iso
✓ Rotação livre para câmera Free
✓ Zoom manual e automático
✓ Reset de posições das câmeras

1.2 Interface do Usuário
-----------------------
✓ Indicador de câmera em tempo real
✓ Controles de zoom na interface
✓ Ajuste de intensidade do skybox
✓ Exibição de informações da câmera atual
✓ Botões de controle intuitivos

1.3 Sistema de Controle
----------------------
✓ Atalhos de teclado para troca de câmeras
✓ Controles WASD para câmera livre
✓ Controles Q/E para movimento vertical
✓ Rotação com mouse e setas
✓ Sistema de zoom suave

1.4 Ajustes Automáticos
----------------------
✓ Ajuste de FOV baseado no tamanho
✓ Ajuste de distância automático
✓ Zoom proporcional ao leito
✓ Manutenção de posições verticais
